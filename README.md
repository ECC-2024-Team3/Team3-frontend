## 기능 
1. 회원가입 : 이메일, 비밀번호, 이름 입력
2. 로그인 : 이메일, 비밀번호 입력
3. 메인 페이지: 상품 나열
4. 상품 등록 페이지
5. 상품 상세 페이지
: 상품 정보, 댓글
6. 마이페이지
: 내 정보 관리, 내 게시물 관리, 내 댓글 관리
